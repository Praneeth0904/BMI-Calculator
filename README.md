## 📌 BMI Calculator

 A simple tool to calculate Body Mass Index based on user input for height and weight, with instant health category feedback.

---

## 🛠️ Tech Stack

HTML5 • CSS3 • JavaScript

---

## ✨ Features

* Calculates Body Mass Index using user-provided height and weight inputs.
* Provides instant feedback on health categories like underweight, normal, or overweight.
* Built with simple form handling and real-time result display.
